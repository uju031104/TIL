# 포인터 변수에서 Const 사용(Const Pointer)

<br/>

C++에서 포인터와 const를 함께 사용할 때는 const 키워드의 위치가 핵심이다.   
Asterisk(*) 기호 앞에 있으면 포인터가 가리키는 데이터가 상수화(값 변경 불가)되고, 뒤에 있으면 포인터 자체가 상수화(주소 변경 불가)된다.   
이를 활용해서 데이터와 포인터 모두 불변으로 설정할 수도 있다. 

### const 위치에 따른 포인터 특징

const int* ptr; (데이터 상수화)   
포인터 ptr을 통해 가리키는 int 값을 변경할 수 없다.   
특징: ptr이 가리키는 주소는 바꿀 수 있다.(다른 변수를 가리킬 수 있음)   
참고: int const* ptr; 와 같다.   

int* const ptr; (포인터 자체 상수화).  
의미: 포인터 ptr이 저장한 주소 값을 변경할 수 없다.   
특징: ptr이 가리키는 주소는 고정되지만, *ptr을 통해 그 주소에 있는 데이터는 변경할 수 있다.   

const int* const ptr; (데이터 & 포인터 모두 상수화).  
의미: 가리키는 데이터도 변경 불가, 포인터의 주소도 변경 불가능하다.
