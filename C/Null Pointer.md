# NULL 포인터(NULL Pointer)

<br/>

#define NULL `((void*)0)`   
아무것도 가르키지 않는 포인터

<br/>

***

<br/>

### NULL 포인터의 쓰임새

<br/>

>포인터의 초기화   
포인터가 더이상 사용중이지 않음을 알리고 싶을 때   
포인터가 유효한 메모리 주소를 저장하고 있는지 확인할 때

<br/>

***

<br/>

### 자료형이 정해지지 않은 포인터

<br/>

>NULL 포인터는 메모리주소값이 아직 정해지지 않았을 때 사용   
자료형을 지금 바로 정할 수 없을 때도 있다.

<br/>

***

<br/>

### void 포인터

<br/>

>범용 포인터라고도 부른다.   
void* 변수명 == 메모리 주소 값;   

어떤 자료형의 포인터라도 void*에 대입 가능

<br/>

***

<br/>

### void 포인터의 주의점

<br/>

>void*에 역참조 연산은 불가능   

해당 메모리 주소부터 몇 바이트만큼 읽어야 내 데이터인지 모르기 때문   

>void*에 정수를 더하거나 빼는 연산은 불가능   

해당 메모리 주소부터 몇 바이트만큼 더하거나 빼야하는지 모르기 때문
