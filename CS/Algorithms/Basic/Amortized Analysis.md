# 분할 상환 분석(Amortized Analysis)

<br/>

분할 상환 분석(Amortized Analysis)은 최악의 경우(Worst-case) 성능이 매우 나쁜 연산이 드물게 발생할 때, 일련의 전체 연산 비용을 평균화하여 실제적인 효율성을 분석하는 알고리즘 분석 기법이다.   
몇 번의 고비용 연산이 대부분의 저비용 연산에 의해 '분할 상환'되어, 전반적인 시간 복잡도를 더 정확하게 파악할 수 있게 한다.  

간단히 말해서 평균적인 시간 복잡도 분석 방법   
특히 스택 기반 알고리즘에서 유용하다.   


### 분할 상환 분석의 의의   

<br/>

**과도한 평가절하 방지**: 최악의 경우(예: 동적 배열의 공간 재할당)만 고려하면 알고리즘이 실제보다 훨씬 느리게 평가될 수 있는데, 분할 상환 분석은 이를 방지합니다.   
**비용의 분산**: 일시적인 고비용 연산이 드물게 발생함을 고려하여, 실제 소요되는 자원(시간, 메모리)을 현실적으로 측정합니다.
